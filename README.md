ほぼ自分用に作成した、AviUtl,AviUtl2で使用できるプラグインです。
指定したフレーム間隔で連番のpng出力が可能です。
連番とは言いますが、ファイル名の末尾に出力フレームがくっつきます。

AviUtlで作成したプレゼン資料を画像化して、パワポに貼り付けるときにでも使ってください。

使い方
普通にプラグインとして読み込んでください。
具体的には、1ではPlugins直下に.auo と .iniを両方置いてください。
2ではC:\ProgramData\aviutl2\Pluginとかにプラグインが入っているはずなので、そこ直下で置くか子ディレクトリを作ってまとめて入れてもいいです。
2じゃないと子ディレクトリ読めないっぽい（でも他のプラグインで子ディレクトリに入れても動くやつあるよね……？　どうやってんのあれ）
